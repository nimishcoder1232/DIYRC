
# DIYRC

This is the repo for an RC transmitter and receiver. The transmitter can operate by itself. The receiver is designed to plug into any aircraft flight computer, using a UART connection.

## Operation

* The controller/transmitter is designed to be plugged in via USB. Preferably a laptop for data stream and troubleshooting. 

* The receiver needs to be powered by the aircraft it is connected to, via 3.3v.

* The receiver requires a custom flight computer that has a UART connection. 

## Radio

* This craft uses a 915 MHz downlink with a LORA radio. This will allow for exceptional range, but no video communication.
* Make sure you pair the Radios on the right frequency and channel to allow for communication.

# SAFETY
#### If this is flown on an aerial vehicle, there are some safety measures required. 

* PLEASE wear safety goggles regardless of how your craft is making thrust. In particular, propellers are DANGEROUS. When testing, stay far, and please wear safety goggles.
* REGISTER YOUR DRONE: Most DIY drones weigh more than 250 grams. Please check your local area for regulations.
